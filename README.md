# Data Cleaning & Wrangling in Excel
Cleaned and transformed raw dataset into a structured, analysis-ready format
Removed duplicate records and handled missing values using Excel filtering and logical functions
Standardized inconsistent text fields and corrected spelling errors
Eliminated excess whitespace using Find & Replace
Reconstructed improperly split columns using Flash Fill
Applied data validation and formatting techniques to improve data quality and consistency

This project demonstrates hands-on experience with data preprocessing, a critical step in the data analysis pipeline.
