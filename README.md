# Data Cleaning & Wrangling in Excel

## Project Description
This project focuses on cleaning and transforming a raw dataset into a structured, analysis-ready format using Microsoft Excel.

## Key Tasks Performed
- Cleaned and transformed raw dataset into a structured format  
- Removed duplicate records and handled missing values  
- Standardized inconsistent text fields and corrected spelling errors  
- Eliminated excess whitespace using Find & Replace  
- Reconstructed improperly split columns using Flash Fill  
- Applied data validation and formatting techniques  

## Tools & Techniques
- Microsoft Excel  
- Data Cleaning & Wrangling  
- Filtering & Sorting  
- Remove Duplicates  
- Conditional Formatting  
- Find & Replace  
- Flash Fill  
- Basic Excel Functions (IF, ISBLANK)  

## Outcome
The final dataset is clean, consistent, and ready for analysis, demonstrating practical data preprocessing skills used in real-world workflows.
